[http://blog.edenmsg.com/angular2-typescript-gulp-and-expressjs/](http://blog.edenmsg.com/angular2-typescript-gulp-and-expressjs/)

Getting Started, grab the dependencies with:

`npm install`

then run

`gulp`

To run the server do this:

```shell
node dist/server.js
```
